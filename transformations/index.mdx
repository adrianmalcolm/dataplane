# Transformations

Transformations are a powerful way to manipulate data returned by a query before the system applies a visualization or perform any other operation.
With transformations,you can manipulate data/structure of the data frames. Some of the examples of the transformations are

- [Limit](./limit) ( ID : `limit` )
- [Merge](./merge) ( ID : `merge` )
- Join time series data
- Join by field
- Join by labels
- Label to fields
- Prepare time series
- Organize fields
- Add field from calculation
- Convert field type
- Reduce
